# 一起记 Couple Ledger

收入驱动型财务管理 + 记账 Web App，面向打工人情侣、家庭和个人。

核心主线：

发工资 -> 开始周期 -> 花钱 -> 看剩多少 -> 撑到下次发工资

## 运行

```bash
npm start
```

打开 http://localhost:4173/。

## 设计系统

- 视觉：深空黑、深墨蓝、暖金、浅金、收入绿、支出橙。
- 字体：SF Pro Display / 系统 UI 字体。
- 组件：8px 圆角、深色玻璃卡片、暖金按钮、绿色进度、金色线框图标。
- 品牌：双圆交叠发光 logo，表达共同账户、共同目标、一起成长。

## 核心功能

- 首页：还能花多少、距离发薪几天、每日可花、已花/预算、存钱率。
- 收入系统：工资、副业、奖金、其他收入，支持发薪日与副业目标。
- 工资周期：默认按发薪日计算周期，例如 04.10 - 05.09。
- 预算系统：预算 = 收入 - 目标存款。
- 记账：支出、收入、转账，字段包含 book_id、member、payer、owner、is_advance。
- 情侣/家庭：个人账本与家庭账本快速切换，共同账户、成员记录、垫付机制。
- 资产：账户余额、资产、负债、净资产、支出结构与趋势。
- 自动化预留：OCR、快捷指令、Siri、周期账单、小组件。

## 部署

这是零依赖静态应用，可以免费部署到 Cloudflare Pages。

部署目录使用仓库根目录即可，构建命令留空，输出目录留空或使用 `/`。
